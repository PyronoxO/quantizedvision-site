# Publishing Constraints

## Required Output Fields
- `sourceId`
- `rewriteTitle`
- `rewriteExcerpt`
- `rewriteBody`
- `approval`

## Length Targets
- Title: 8-14 words
- Excerpt: 25-45 words
- Body: 120-220 words

## Workflow Status
- Imported posts start as `review`.
- Only manually published posts go live.

## Formatting
- Plain text, no markdown tables unless requested.
- Keep paragraphs short (2-4 total).
- If output is CSV, keep one row per article and preserve `sourceId`.
