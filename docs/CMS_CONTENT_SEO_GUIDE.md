# CMS Content SEO Guide (Landing Pages)

Use this for `/about`, `/posts`, `/projects`, `/gallery` page entries in Sanity.

## Meta title pattern
- Keep between 50-60 chars.
- Use intent + brand.
- Examples:
  - `AI Artist Portfolio and Visual Worlds | Quantized Vision`
  - `AI Projects and Motion Experiments | Quantized Vision`

## Meta description pattern
- 120-160 chars, plain language, no keyword stuffing.
- Start with what the page contains.
- Mention one outcome or value.

## Hero body
- First paragraph should clearly state page intent.
- Keep 2-3 short paragraphs max.
- Include terms naturally:
  - AI art
  - visual storytelling
  - motion
  - digital media

## Internal linking
- Every landing page should link to at least 2 other core sections:
  - `/posts`
  - `/projects`
  - `/gallery`
  - `/about`

## Publishing cadence
- Keep fresh activity in posts/projects.
- New URLs improve crawl frequency and recommendation eligibility.
